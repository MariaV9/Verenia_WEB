Тег <ol>

Тег <ol> встановлює нумерований список. Кожний елемент списку повинен починатися з тегу <li>.
Якщо до тегу <ol> застосовується таблиця стилів, то елементи <li> успадковують ці властивості.

Синтаксис

<ol>

<li> елемент нумерованого списку <li>

<li> елемент нумерованого списку <li>

</ol>

Атрибути

Атрибут Призначення
type Встановлює вид маркера списку
reversed Нумерація в списку стає по зменшенню
start Задає число, з якого буде починатися нумерований список
