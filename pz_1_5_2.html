Призначення Назва використовуваного тегу Короткий опис Стандарт
Для каркасу документа < html > Вказує програмі перегляду сторінок, що це HTML документ HTML5
< head > Визначає місце, де зберігається різноманітна інформація, що не відображається в тілі документа. Тут розташовується тег назви документа та теги для пошукових машин HTML5
< body > Визначає видиму частину документа HTML5
< title > Поміщує назву документа в зміст програми перегляду сторінок HTML5
Теги форматування тексту < pre > Вставляє попередньо отформатований текст HTML5
< b > Створює жирний текст HTML5
< i > Створює текст курсивом HTML5
< em > Використовується для виділення з тексту слова(курсив або жирний текст) HTML5
< p > Визначає текстовий абзац HTML5
< strong > Призначений для акцентування тексту HTML5
< br > Встановлює новий рядок в тому місці, де цей тег знаходиться HTML5
< hr > Створює горизонтальну лінію, яка за своїм виглядом залежить від використовуваних параметрів, а також браузера HTML5
< ul > Встановлює маркований список HTML5
< i > Встановлює курсивне оформлення шрифту HTML5
< li > Визначає окремий елемент списку HTML5
< ol > Встановлює нумерований список HTML5
< u > Додає підкреслення до тексту HTML5
< s > Відображає текст як перекреслений HTML5
Для гіперпосилань <a> Створює гіперсилку на інші документи або частину поточного документа HTML5
Графічні елементи < img src ="name" > Вставляє попередньо отформатований текст HTML5
< img src="name" align=? > Атрибут ALIGN тегу IMG встановлює режим вирівнювання зазначеного зображення HTML5
< img src="name" border=? > Встановлює режим відображення рамки зазначеного зображення HTML5
